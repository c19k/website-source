This project is maintained and hosted in GitHub. The dataset is managed with Google Sheets. Kerala GeoJSON is generously provided by Sajjad Anwar.

This project is a clone of https://covid19japan.com/. Built and maintained by Shane Reustle (Reustle K.K.) and many other contributors. Original concept and design by Jiahui Zhou.

To maintain adaptability with the progress of the original project, some of the source code and data facets are kept as it is. It will be customized and localized in the coming days.
